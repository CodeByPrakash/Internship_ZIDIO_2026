# IntellMeet 🧠

> AI-Powered Enterprise Meeting & Collaboration Platform by Zidio Development

## Overview
IntellMeet transforms every meeting into an actionable, trackable event using real-time video, AI-powered transcription, smart action items, team chat, and analytics — reducing meeting follow-up time by 40–60%.

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React 19 + TypeScript + Vite |
| UI | shadcn/ui + Tailwind CSS v4 |
| State | TanStack Query + Zustand |
| Backend | Node.js + Express + TypeScript |
| Database | MongoDB + Mongoose |
| Real-Time | Socket.io + WebRTC |
| AI | OpenAI + HuggingFace |
| Cache | Redis |
| Auth | JWT + bcrypt |
| Storage | Cloudinary / AWS S3 |

## Project Structure
```
intellmeet/
├── client/    # React frontend
└── server/    # Express backend
```

## Getting Started

### Prerequisites
- Node.js 20+
- MongoDB 7+
- Redis 7+

### Server
```bash
cd server
npm install
cp .env.example .env   # fill in your values
npm run dev
```

### Client
```bash
cd client
npm install
npm run dev
```

## API Base URL
`http://localhost:5000/api`
